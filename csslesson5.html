Animations and Transition

CSS Animations can be very hard but make very cool looking website. With animations it is also imported to talk about opacity, cursor, and transition.

Also display and icon can change the style and outline of your html pages.

CSS Animations


An animation lets an element gradually change from one style to another and you can change the CSS properties as many times as you want.

Keyframes hold what style the element will have at a certain time so to use CSS animation, you must first specify some keyframes for the animation.

Animation-name is how keyframes knows what to add to where. The animation-duration defines how long an animation should take to complete but if the animation-duration is not specified, no animation will occur because the default value is 0 seconds. The animation-iteration-count property specifies the number of times an animation should run. The animation-delay property specifies a delay for the start of an animation. The animation-timing-function property specifies the speed curve of the animation.

@keyframes example {

from {background-color: red;}

to {background-color: yellow;}

}

div {

width: 100px;

height: 100px;

background-color: red;

animation-name: example;

animation-duration: 6s;

animation-iteration-count: infinite;

animation-timing-function: ease;

animation-delay: 2s;

}

CSS Transition


CSS transitions allows you to change property values smoothly and over a given duration. The transition property is best used with Pseudo-Classes.

To create a transition effect, you must specify two things the first is that the CSS property you want to add an effect to and the duration of the effect.

div {

width: 100px;

height: 100px;

background: green;

transition: width 2s;

}

div:hover {

width: 300px;

}
