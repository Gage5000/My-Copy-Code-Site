HTML Media

media comes in many different formats. It can be almost anything you can hear or see, like images, music, sound, videos, records, films, animations, and more.

In HTML there are a lot of ways to display media like the video, audio, iframe and img.

The Video Tag

The <video> tag defines a video using src gets the video.

Some attribute like autoplay, controls, and loop change the amout of control the user has over the video.


<video src="REPLEACE" controls><video>

The Audio Tag

The <audio> tag defines a audio player.


<audio src="REPLEACE" controls></audio>

The iframe Tag

The <iframe> tag enbeds website, image, or youtube videos.


<iframe src="REPLACE"></iframe>
