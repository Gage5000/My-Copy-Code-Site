CSS Pseudo-Classes and Elements

Pseudo-classes are used to define a special state of an element. Examples of a special state can be styling an element when a user moves over it, styling visited and unvisited links differently, or styling an element when it gets focused.

Pseudo-elements are used to style specific parts of an element. Examples of specific parts can be styling the first letter or line, of an element, insert content before or after an element, or styling the markers of list items.


CSS Links Anchor Pseudo-Classes


Link is the base color of an A tag. Only works for the A tag.

Visited changs the color if you click on the A tag changing it permanently. Only works for the A tag.

Hover changs the color if you hover over the tag but if you take the corser off it stops changing it. Works with all tags.

Active only work of the small time you click on the tag. Works with all tags.

a:link {

color: red;

}

a:visited {

color: green;

}

a:hover {

color: hotpink;

}

a:active {

color: blue;

}

First-Child Pseudo-Classes


The first-child pseudo-class matches a specified element that is the first child of another element.

p:first-child {

First-Line Pseudo-Element


The first-line adds the style to the first line of any text tags but nothing alse after the line.

Pseudo-Element use two :: not like pseudo-classes that only use one :.

p::first-line {

color:darkmagenta;

}

This is the 1st P tag. But This text is just here to show how this First-line Pseudo-Element Works.
And as you can see this text dose not have the color.


First-Letter Pseudo-Element


The first letter pseudo-element makes the first letter styled but nothing alse after the letter.

p::first-letter {

color: lightgreen

}

The T in this the is blue because it is the first letter.


Before and After Pseudo-Element


The before pseudo-element adds an element to the start of a another element.

The after pseudo-element adds an element to the end of a another element.

p::before {

content: url("gray.png");

}

p::after {

content: url("gray.png");

}

This is the first p tag


Marker Pseudo-Element


The marker pseudo-element selects the markers of list items and styles the markers.

li::marker {
