Text Formatting and Display

There are a lot of ways to format text using fonts, align, tables, and lists.

CSS Text Align

The text-align property is used to set the horizontal alignment of a text.

The text can be left, right, centered, or justified aligned.


h1 {

text-align: center;

}

h2 {

text-align: left;

}

h3 {

text-align: right;

}

CSS Fonts

  • Font-family property to specify the font of a text. The font-family property should hold several font names as fallbacks to ensure compatibility between browsers and operating systems. Start with the font you want, and end with a generic family.
  • Font-style property is mostly used to specify italic text.
  • Font-weight property specifies the weight of a font like if it is bold or not.
  • Font-variant property specifies text should be displayed in a small-caps font. In a small-caps font, all lowercase letters are converted to uppercase letters. However, the converted uppercase letters appears in a smaller font size than the original uppercase letters in the text.
  • Font-size property can change the size of the text.

h2 {

font-family: 'Times New Roman', 'Hoefler Text', 'Liberation Serif', Times, Cambria, 'serif';

}

h3 {

style="font-style: italic;

}

h4 {

style="font-weight: bold;

}

h5 {

font-variant: small-caps;

}

h6 {

font-size: 50px;

}

CSS Float

The float property is used for positioning and formatting content like let text float left or right.

The float property can have one of the following values

  • left - The element floats to the left of its container
  • right - The element floats to the right of its container
  • none - The element does not float, this is default
  • inherit - The element inherits the float value of its parent

.pInherit {

float: inherit;

}

.pNone {

float: none;

}

.pLeft {

float: left;

}

.pRight {

float: right;

}
