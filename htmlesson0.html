What Is HTML?

HTML stands for HyperText Markup Language.

You can write HTML on a lot of different services like Dreamweaver, Notepad ++, and Visual Studio for just some examples.

Other programs can also do stuff with html like, Cascading Style Sheets (CSS) and JavaScript (JS).

HTML is the standard markup language for creating Web pages

HTML describes the structure of a Web page and tells the browser how to display the content.


What Does It Look Like In a HTML Page?

The start of a HTML doc must start with <!doctype html> because this tells the browser what type of document to expect. If you don't use the doctype the it can cause the browser to render the page differently than intended.

Most tags have a start and an end with the end having a slash like so <Example></Example>.

But there are some that do not have a end tag like <br>. The <br> tag defines a line break, and is an empty element without a closing tag.

The <html></html> tag tells the browser that the code for the web page starts. All html code must be within the tag.

The <head></head> tag contains all of the information about the web site also known as metadata, which is simply data about data. This means information in the head tag is not displayed on the page but the information is used by the browsers and search engines.

A good example of metadata is the <title></title> tag which is display name of the site. For This means best buys title tag is <title>Best Buy | Official Online Store | Shop Now and Save</title>.

The <body></body> tag is all of the code that the user can see and is were most of the coding is done.

The start of a html page shode look like this.

<!doctype html>

<html>

<head>

<title>THIS IS THE NAME OF THE PAGE IN THE WEBSITE.</title>

</head>

<body>

</body>

</html>
