What Is CSS?

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

CSS can be in a external file, Internal using the style tag, or Inline using the style attribute.

The Right Syntax for CSS


The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

body {

background-color: lightblue;

}

h1 {

color: white;

text-align: center;

}

p {

font-family: verdana;

font-size: 20px;

}


CSS Selectors


CSS selectors are used to find and select HTML elements you want to style.

There are five ways to select elements by id, class, *, the element or mlti elements

  • Id selects the one element with id="blank". There can only be one of each id.
  • Class selects all elements with class="blank". There can be a lot of one class.
  • * selects all elements. ALL.
  • Selecting all of one element like p or h1.
  • Mlti elements selects more than one element and applies the CSS properties to them.

<style>

#idColor {

color: red;

}

.classColor {

color: green;

}

* {

background-color: pink;

}

p {

color:blue;

}

h1, h2, h3 {

color:yellow;

}

</style>

CSS Comments


CSS comments are not displayed on the page, but they can help document your source code.

Comments are used to explain the code, and may help when you edit the source code at a later date.

CSS !important Rule


The !important rule in CSS is used to add more importance to a property/value than normal.

Meaning if you use the !important rule, it will override ALL previous styling rules for that specific property on that element, regardless of id or class.

#idColor {

background-color: blue;

}

.classColor {

background-color: green;

}

p {

background-color: red !important;

}

This is the color.


CSS Combinators


A combinator is something that explains the relationship between the selectors.

A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.

There are four different combinators in CSS.

  • Descendant combinator (space)
  • Child combinator (>)
  • Next sibling combinator (+)
  • Subsequent-sibling combinator (~)

div p{

color: blue;

}

div > p{

color: blue;

}

div + p{

color: blue;

}

div ~ p{

color: blue;

}
