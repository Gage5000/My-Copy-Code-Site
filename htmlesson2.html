Text Formatting Tags

HTML has several tags for defining text with a special meaning.

All text HTML elements can be affected by Formatting tags if you put them around the text tag like so: <b><p>Text</p></b>.

but you also can just use the formatting tags by themself.

The B Tag

The <b> tag defines the text within it as bold text making all of the text bold.


<b>Bolded Text<b>

<p>Non-bolded Text<p>
Bolded Text

Non-bolded Text


The Strong Tag

The <strong> element defines text with importance and content inside is displayed bolded.

The different between the B and Strong tags are that strong is more important than the B tag.


<strong>Strong Text</strong>

<p>Non-Strong Text</p>
Strong Text

Non-Strong Text


The I Tag

The <i> tag makes text italic.


<i>Italic Text</i>

<p>Non-italic Text</p>
Italic Text

Non-italic Text


The Em Tag

The <em> tag defines emphasized text and the content inside is displayed in italic.

Just like bolded the only different between the the two tags is the use case with the em tag used if you need text emphasized and the I tag if you just need text italic.


<em>Emphasized Text</em>

<p>Non-emphasized Text</p>
Emphasized Text

Non-emphasized Text

<small>Smaller Text</Small>

The Del Tag

The <del> tag defines text that has been deleted from the page and the browsers usually strike a line through the deleted text.

The Sub Tag

The <sub> tag is subscript text that is half a character below the normal size.


<p>Non-subscript Text<sub>Subscript Text</sub></p>

The Sup Tag

The <sup> tag defines superscript text it is half a character above the normal line.


<p>Non-subscript Text<sup>Subscript Text</sup></p>

HTML Block and Inline Tags

Block-level Tags always starts on a new line, and the browsers automatically adds some space (margin) before and after the element.

A block-level element always takes up the full width available.

Here are all of the block-level tags:<address><article><aside><blockquote><canvas><dd><div><dl><dt><fieldset><figcaption><figure><footer><form><h1><h2><h3><h4><h5><h6><header><hr><li><main><nav><noscript><ol><p><pre><section><table><tfoot><ul><video>


Inline-level are the opposite of block tags meaning it does not start on a new line and only takes up as much width as necessary.

Here are all of the Inline-level tags: <a><abbr><acronym><b><bdo><big><br><button><cite><code><dfn><em><i><img><input><kbd><label><map><object><output><q><samp><script><select><small><span><strong><sub><sup><textarea><time><tt><var>
