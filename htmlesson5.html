Form and Input Tags

A form is used to collect user input and most of the time the users inputs are sent to a server for processing.

The form tag is a container for different types of input elements like text fields, checkboxes, radio buttons, submit buttons, and more.

The Form Tag

The <form> tag is used to create an space for users to input info.

The placeholder is used to tell the user what the input wants.


<form>

<input placeholder="What is the best number?">

<input placeholder="What is the best color?">

<input placeholder="What is the best food?">

<input placeholder="What is the best HTML tag?">

</form>

The Input Tag

The <input> tag can get info for a lot of different uses.

It can be displayed in many ways, depending on the type attribute.

The bace input alows you to put words or info into the input.


<input placeholder="Click Me And Say Hi">

Type Text

The text type defines a single-line input field for text input.

As you can see this is the same as the one with no type.


<input placeholder="Click Me And Say Hi" type="text">

Type Checkbox

The checkbox type tag makes a box that you can check to see input of yes or no.


<label>do you like HTML?</label>

<input type="checkbox">

<label>Do you like CSS?</label>

<input type="checkbox">

<label>Do you like JS?</label>

<input type="checkbox">

<label>Do you like C++?</label>

<input type="checkbox">

Type Submit

The submit type makes a button that has the word submit in it.


<input type="submit">

Type Number

The number type makes it so that you can only add number to the input.

This incleds E and dots.


<input value="384.93e45" type="number">

Type Password

The password type makes all of the word in it dots.


<input value="YOUARENOTSEEINGTHISIHOPE!" type="password">

Type Color

The color type makes a color picker.


<input value="#0266ff" type="color">

Type Date

The date type makes a calinder.


<input type="date">
